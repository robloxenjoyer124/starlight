## ğŸ› ï¸ Configuration

Create a `.rofinderrc` or `config.yml` in your home directory:

```yaml
# API Configuration
api:
  base_url: "https://api.roblox.com"
  timeout: 15
  retries: 3
  rate_limit: 60  # requests per minute
  user_agent: "RoFinder/2.1.0"

  format: "table"       # table, json, csv, minimal, detailed
  theme: "default"      # default, cyberpunk, matrix, neon, etc.
  colors: true
  unicode: true
  timestamps: true
  show_tips: true

  enabled: true
  directory: "~/.rofinder/cache"
  ttl: 3600           # 1 hour
  max_size: 100       # MB

  parallel_requests: true
  max_workers: 5
  compress_output: false
  log_level: "INFO"   # DEBUG, INFO, WARNING, ERROR
  proxy: null         # http://proxy.example.com:8080

  save_history: true
```
